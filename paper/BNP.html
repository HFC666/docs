    # Introduction

    # Construction

    那我们如何构建GGG呢？即如何从HHH中采样得到GGG呢？我们采用stick-breaking construction的方法来产生GGG。其构建方式为：

    1. 从HHH中采样得到θ1\theta_1θ​1​​，即θ1∼H\theta_1\sim Hθ​1​​∼H
    2. 采样β1∼Beta(1,α)\beta_1\sim\text{Beta}(1,\alpha)β​1​​∼Beta(1,α)

    6. 权重π2=(1−π)β2\pi_2 = (1-\pi)\beta_2π​2​​=(1−π)β​2​​ 以此类推，这样权重π2\pi_2π​2​​相当于是从取完π1\pi_1π​1​​剩下的权重中取得。

    我们在看一下关于α\alphaα得一些内容，当α=0\alpha=0α=0时，E(βi)=1,Var(βi)=0\mathbb{E}(\beta_i)=1,\text{Var}(\beta_i)=0E(β​i​​)=1,Var(β​i​​)=0，所以一次就把权重全部采完，即只产生一个样本，对应于最离散的情况，而当α=∞\alpha=\inftyα=∞时，E(βi)=0\mathbb{E}(\beta_i)=0E(β​i​​)=0，相当于连续分布的情况。我们常将此采样方法写为π∼GEM(α)\pi \sim \text{GEM}(\alpha)π∼GEM(α)。

    # Predictive distribution

    什么是预测分布呢？预测分布为：

    P(Xi∣X−i)=∫wP(Xi,w∣X−i)dw=∫wP(Xi∣w,X−i)P(w∣X−i)dw=∫wP(Xi∣w)P(w∣X−i)dw\begin{aligned} P(X_i\mid X_{-i}) &= \int_w P(X_i,w\mid X_{-i})dw\\ &= \int_w P(X_i\mid w,X_{-i})P(w\mid X_{-i})dw\\ &= \int_wP(X_i\mid w)P(w\mid X_{-i})dw \end{aligned} ​P(X​i​​∣X​−i​​)​​​​​=∫​w​​P(X​i​​,w∣X​−i​​)dw​=∫​w​​P(X​i​​∣w,X​−i​​)P(w∣X​−i​​)dw​=∫​w​​P(X​i​​∣w)P(w∣X​−i​​)dw​​

    P(θi∣θ−i)=∫GP(θi∣G)P(G∣θ−i)dGP(\theta_i\mid \theta_{-i}) = \int_G P(\theta_i\mid G)P(G\mid \theta_{-i})dG P(θ​i​​∣θ​−i​​)=∫​G​​P(θ​i​​∣G)P(G∣θ​−i​​)dG

    θ1,θ2,⋯\theta_1,\theta_2,\cdotsθ​1​​,θ​2​​,⋯预测分布的序列被称为Blackwell-MacQueen urn scheme。这个名字来源于一个隐喻。特别地，在Θ\ThetaΘ中的每一个值都是唯一的颜色，并且抽样θ∼G\theta\sim Gθ∼G来给球上色。另外我们有一个盒子来装之前看过的球。起初在盒子里没有球，我们从HHH中取颜色，θ1∼H\theta_1\sim Hθ​1​​∼H，给球上色并将其放在盒子里。在之后的步骤中，如在n+1n+1n+1步中，我们要么以αα+n\frac{\alpha}{\alpha+n}​α+n​​α​​抽取一个新颜色(θn+1∼H\theta_{n+1}\sim Hθ​n+1​​∼H)，给球染色并将其放到盒子中，或者以概率nα+n\frac{n}{\alpha+n}​α+n​​n​​从盒子中取出一个球，将新球涂成它的颜色(从经验分布中抽样)并放到盒子里。

    Blackwell-MacQueen urn scheme可以被用来证明DP的存在。我们可以在序列θ1,θ2,⋯\theta_1,\theta_2,\cdotsθ​1​​,θ​2​​,⋯上构建分布，通过迭代地在给定θ1,⋯,θi−1\theta_1,\cdots,\theta_{i-1}θ​1​​,⋯,θ​i−1​​的条件下采样θi\theta_iθ​i​​。对于n≥1n\ge1n≥1令

    P(θ1,⋯,θn)=∏i=1nP(θi∣θ1,⋯,θi−1)P(\theta_1,\cdots,\theta_n) = \prod_{i=1}^nP(\theta_i\mid \theta_1,\cdots,\theta_{i-1}) P(θ​1​​,⋯,θ​n​​)=​i=1​∏​n​​P(θ​i​​∣θ​1​​,⋯,θ​i−1​​)

    可以得到这个随机序列是无限可交换的。也就是说，对于每一个nnn，生成θ1,⋯,θn\theta_1,\cdots,\theta_nθ​1​​,⋯,θ​n​​的概率等于以任何顺序采样得到它们的概率。

    # Dirichlet Process Mixture Models

    狄利克雷混合模型可以写为：

    π∣α∼GEM(α)θk∗∣H∼Hzi∣π∼Mult(π)xi∣zi,{θk∗}∼F(θzi∗)\begin{aligned} &\pi \mid \alpha \sim \text{GEM}(\alpha)\\ &\theta_{k}^{*} \mid H \sim H\\ &z_{i} \mid \pi \sim \text{Mult}(\pi)\\ &x_{i} \mid z_{i},\left\{\theta_{k}^{*}\right\} \sim F\left(\theta_{z_{i}}^{*}\right) \end{aligned} ​​​​​​​π∣α∼GEM(α)​θ​k​∗​​∣H∼H​z​i​​∣π∼Mult(π)​x​i​​∣z​i​​,{θ​k​∗​​}∼F(θ​z​i​​​∗​​)​​

    狄利克雷混合模型为无限混合模型，指的是具有无限可数个类的混合模型。与事先确定了类的有限混合模型不同，狄利克雷混合模型会根据数据确定聚类的数目。

    狄利克雷过程为无参贝叶斯方法的一种，为什么是无参？我们理解是狄利克雷过程是分布的先验，而狄利克雷过程采样得到的分布为我们产生数据的分布，在有参数的模型中此分布有参数(感觉像废话)，而狄利克雷过程产生的分布无法用参数表示故为无参数模型。我们将分布作为概率的一部分，并且存在自己的先验分布，也可以让我们通过数据来自动调节分布的复杂度。
