    # Auto-Encoding Variational Bayes

    术语：

    • Variational Bayesian(VB)：变分贝叶斯
    • Stochastic Gradient Variational Bayes(SGVB)：随机梯度变分贝叶斯
    • Auto-Encoding Variational Bayes(AEVB)：自编码变分贝叶斯

    # Method

    # Problem scenario

    我们的数据集X={x}i=1N\mathrm{X} = \{\mathrm{x}\}_{i=1}^NX={x}​i=1​N​​独立同分布。假设我们的数据是从一些随机过程中产生的，这个过程包含两个步骤：

    • z(i)\mathrm{z}^{(i)}z​(i)​​是从先验分布pθ⋆(z)p_{\theta^\star}(\mathrm{z})p​θ​⋆​​​​(z)中产生的。
    • x(i)\mathrm{x}^{(i)}x​(i)​​是从条件分布pθ⋆(x∣z)p_{\theta^\star}(\mathrm{x\mid z})p​θ​⋆​​​​(x∣z)中产生的。

    我们着重解决下列情况：

    • 不可追踪：在边缘似然pθ(x)=∫pθ(x)pθ(x∣z)dzp_\theta(x) = \int p_\theta(x)p_\theta(x\mid z)dzp​θ​​(x)=∫p​θ​​(x)p​θ​​(x∣z)dz是不可追踪的，比如我们的pθ(x∣z)p_\theta(x\mid z)p​θ​​(x∣z)是一个神经网络。
    • 大数据：我们有很多的数据以至于批量优化很费力，我们将会用小批量甚至单个数据点来更新参数。

    我们将要解决下面三个问题：

    • 对于参数θ\thetaθ的ML或者MAP的高效近似。
    • 选定θ\thetaθ和给定观测变量xxx后隐变量zzz后验推断的高效近似。
    • 变量xxx边缘推断的高效近似。

    为了解决上述问题，我们引入了recognition模型qϕ(z∣x)q_\phi(z\mid x)q​ϕ​​(z∣x)：对不可追踪的真正后验pθ(z∣x)p_\theta(z\mid x)p​θ​​(z∣x)的近似。我们认为qϕ(z∣x)q_\phi(z\mid x)q​ϕ​​(z∣x)是编码器，而pθ(x∣z)p_\theta(x\mid z)p​θ​​(x∣z)是解码器。

    # The variational bound

    因为数据是独立同分布的：

    logpθ(x(1),⋯,x(N))=∑i=1Nlogpθ(x(i))\log p_\theta(x^{(1)},\cdots,x^{(N)}) = \sum_{i=1}^N\log p_\theta(x^{(i)}) logp​θ​​(x​(1)​​,⋯,x​(N)​​)=​i=1​∑​N​​logp​θ​​(x​(i)​​)

    可以被写为：

    我们想要优化L(θ,ϕ;x(i))\mathcal{L}(\theta,\phi;x^{(i)})L(θ,ϕ;x​(i)​​)，需要对其求导，但是ϕ\phiϕ的梯度有一些病态。一般的蒙特卡洛方法来估计其梯度是：

    ∇ϕEqϕ(z)[f(z)]=Eqϕ(z)[f(z)∇qϕ(z)logqϕ(z)]≃1L∑l=1Lf(z)∇qϕ(z(l))logqϕ(z(l))\nabla_\phi\mathbb{E}_{q_\phi(z)}\left[f(z)\right] = \mathbb{E}_{q_\phi(z)}\left[f(z)\nabla_{q_\phi(z)}\log q_\phi(z)\right] \simeq \frac{1}{L}\sum_{l=1}^Lf(z)\nabla_{q_\phi(z^{(l)})}\log q_{\phi}(z^{(l)}) ∇​ϕ​​E​q​ϕ​​(z)​​[f(z)]=E​q​ϕ​​(z)​​[f(z)∇​q​ϕ​​(z)​​logq​ϕ​​(z)]≃​L​​1​​​l=1​∑​L​​f(z)∇​q​ϕ​​(z​(l)​​)​​logq​ϕ​​(z​(l)​​)

    这是怎么来的呢？我们推理一下：

    ∇ϕEqϕ(z)[f(z)]=∇ϕ∫f(z)qϕ(z)dz=∫f(z)∇qϕ(z)dz=∫f(z)qϕ(z)[∇qϕ(z)logqϕ(z)]dz\begin{aligned} \nabla_\phi\mathbb{E}_{q_\phi(z)}[f(z)] &= \nabla_\phi \int f(z)q_\phi(z)dz\\ &= \int f(z)\nabla q_\phi(z)dz\\ &= \int f(z)q_\phi(z)[\nabla_{q_\phi(z)}\log q_{\phi}(z)]dz \end{aligned} ​∇​ϕ​​E​q​ϕ​​(z)​​[f(z)]​​​​​=∇​ϕ​​∫f(z)q​ϕ​​(z)dz​=∫f(z)∇q​ϕ​​(z)dz​=∫f(z)q​ϕ​​(z)[∇​q​ϕ​​(z)​​logq​ϕ​​(z)]dz​​

    但是这种方式存在问题，因为qϕ(z)∈(0,1)q_\phi(z)\in (0,1)q​ϕ​​(z)∈(0,1)，而log\loglog函数在(0,1)(0,1)(0,1)之间的梯度变换很大，这就导致结果有很大的变异性。

    # The SGVB estimator and AEVB algorithm

    z~=gϕ(ϵ,x),withϵ∼p(ϵ)\tilde{z} = g_\phi(\epsilon, x), \text{with } \epsilon \sim p(\epsilon) ​z​~​​=g​ϕ​​(ϵ,x),withϵ∼p(ϵ)

    我们现在可以通过使用蒙特卡洛方法来估计函数f(z)f(z)f(z)的期望：

    Eqϕ(z∣x(i))[f(z)]=Ep(ϵ)[f(gϕ(ϵ,x(i)))]≃1L∑l=1Lf(gϕ(ϵ(l),x(i))),whereϵ(l)∼p(ϵ)\mathbb{E}_{q_\phi(z\mid x^{(i)})}[f(z)] = \mathbb{E}_{p(\epsilon)}\left[f(g_\phi(\epsilon, x^{(i)}))\right] \simeq \frac{1}{L}\sum_{l=1}^L f(g_\phi(\epsilon^{(l)},x^{(i)})), \text{where }\epsilon^{(l)} \sim p(\epsilon) E​q​ϕ​​(z∣x​(i)​​)​​[f(z)]=E​p(ϵ)​​[f(g​ϕ​​(ϵ,x​(i)​​))]≃​L​​1​​​l=1​∑​L​​f(g​ϕ​​(ϵ​(l)​​,x​(i)​​)),whereϵ​(l)​​∼p(ϵ)

    我们将这个技术应用到变分下界，就形成了随机梯度变分贝叶斯(SGVB)估计L~A(θ,ϕ;x(i))≃L(θ,ϕ;x(i))\tilde{\mathcal{L}}^A(\theta,\phi;x^{(i)})\simeq \mathcal{L}(\theta,\phi;x^{(i)})​L​~​​​A​​(θ,ϕ;x​(i)​​)≃L(θ,ϕ;x​(i)​​)：

    L~A(θ,ϕ;x(i))=1L∑l=1Llogpθ(x(i),z(i,l))−logqϕ(z(i,l)∣x(i))wherez(i,l)=gϕ(ϵ(i,l),x(i))andϵ(l)∼p(ϵ)\begin{aligned} &\tilde{\mathcal{L}}^A(\theta,\phi;x^{(i)}) = \frac{1}{L}\sum_{l=1}^L\log p_\theta(x^{(i)},z^{(i,l)}) - \log q_\phi(z^{(i,l)}\mid x^{(i)})\\ &\text{where }z^{(i,l)} = g_\phi(\epsilon^{(i,l)},x^{(i)})\text{ and }\epsilon^{(l)}\sim p(\epsilon) \end{aligned} ​​​​​​L​~​​​A​​(θ,ϕ;x​(i)​​)=​L​​1​​​l=1​∑​L​​logp​θ​​(x​(i)​​,z​(i,l)​​)−logq​ϕ​​(z​(i,l)​​∣x​(i)​​)​wherez​(i,l)​​=g​ϕ​​(ϵ​(i,l)​​,x​(i)​​)andϵ​(l)​​∼p(ϵ)​​

    一般，KL散度KL(qϕ(z∣x(i))∥pθ(z))\mathbb{KL}(q_\phi(z\mid x^{(i)})\| p_\theta(z))KL(q​ϕ​​(z∣x​(i)​​)∥p​θ​​(z))的积分是有解析解的，因此只有Eqϕ(z∣x(i))[logpθ(x(i)∣z)]\mathbb{E}_{q_\phi(z\mid x^{(i)})}[\log p_\theta(x^{(i)}\mid z)]E​q​ϕ​​(z∣x​(i)​​)​​[logp​θ​​(x​(i)​​∣z)]需要通过采样来估计。我们可以把KL\mathbb{KL}KL项看作是对ϕ\phiϕ的正则化。这就形成了SGVB的另一个版本：

    L~B(θ,ϕ;x(i))=−KL(qϕ(z∣x(i))∥pθ(z))+1L∑l=1L(logpθ(x(i)∣z(i,l)))wherez(i,l)=gϕ(ϵ(i,l),x(i))andϵ(l)∼p(ϵ)\begin{aligned} \tilde{\mathcal{L}}^B(\theta,\phi;x^{(i)}) &= -\mathbb{KL}(q_\phi(z\mid x^{(i)})\| p_\theta(z)) + \frac{1}{L}\sum_{l=1}^L(\log p_\theta(x^{(i)}\mid z^{(i,l)}))\\ \text{where }z^{(i,l)} &= g_\phi(\epsilon^{(i,l)},x^{(i)})\text{ and }\epsilon^{(l)}\sim p(\epsilon) \end{aligned} ​​L​~​​​B​​(θ,ϕ;x​(i)​​)​wherez​(i,l)​​​​​=−KL(q​ϕ​​(z∣x​(i)​​)∥p​θ​​(z))+​L​​1​​​l=1​∑​L​​(logp​θ​​(x​(i)​​∣z​(i,l)​​))​=g​ϕ​​(ϵ​(i,l)​​,x​(i)​​)andϵ​(l)​​∼p(ϵ)​​

    # The reparameterization trick

    重参数化的技巧很简单。令zzz为连续随机变量，z∼qϕ(z∣x)z\sim q_\phi(z\mid x)z∼q​ϕ​​(z∣x)。我们可以将随机变量zzz表述为确定变量z=gϕ(ϵ,x)z = g_\phi(\epsilon,x)z=g​ϕ​​(ϵ,x)，其中ϵ\epsilonϵ是具有独立的边缘分布p(ϵ)p(\epsilon)p(ϵ)的辅助变量，gϕ(⋅)g_\phi(\cdot)g​ϕ​​(⋅)是以ϕ\phiϕ为参数的向量值函数。

    这样做的目的是将以qϕ(z∣x)q_\phi(z\mid x)q​ϕ​​(z∣x)为概率密度函数的期望重写使得对期望的蒙特卡洛估计对ϕ\phiϕ是可导的。

    证明如下：

    给定确定的映射z=gϕ(ϵ,x)z = g_\phi(\epsilon,x)z=g​ϕ​​(ϵ,x)，我们知道：

    qϕ(z∣x)∏idzi=p(ϵ)∏idϵiq_\phi(z\mid x)\prod_idz_i = p(\epsilon)\prod_id\epsilon_i q​ϕ​​(z∣x)​i​∏​​dz​i​​=p(ϵ)​i​∏​​dϵ​i​​

    因此：

    ∫qϕ(z∣x)f(z)dz=∫p(ϵ)f(z)dϵ=∫p(ϵ)f(gϕ(ϵ,x))dϵ\int q_\phi(z\mid x)f(z)dz = \int p(\epsilon)f(z)d\epsilon = \int p(\epsilon)f(g_\phi(\epsilon,x))d\epsilon ∫q​ϕ​​(z∣x)f(z)dz=∫p(ϵ)f(z)dϵ=∫p(ϵ)f(g​ϕ​​(ϵ,x))dϵ

    一个可微分的估计可以写作：

    ∫qϕ(z∣x)f(z)dz≃1L∑l=1Lf(gϕ(x,ϵ(l))),ϵ(l)∼p(ϵ)\int q_\phi(z\mid x)f(z)dz\simeq \frac{1}{L}\sum_{l=1}^L f(g_\phi(x,\epsilon^{(l)})),\quad \epsilon^{(l)}\sim p(\epsilon) ∫q​ϕ​​(z∣x)f(z)dz≃​L​​1​​​l=1​∑​L​​f(g​ϕ​​(x,ϵ​(l)​​)),ϵ​(l)​​∼p(ϵ)

    例如，单变量高斯例子：令z∼p(z∣x)=N(μ,σ2)z \sim p(z\mid x) = \mathcal{N}(\mu,\sigma^2)z∼p(z∣x)=N(μ,σ​2​​)。在此情况下，一个重参数化是z=μ+σϵz = \mu + \sigma\epsilonz=μ+σϵ，其中σ\sigmaσ为辅助噪声变量ϵ∼N(0,1)\epsilon \sim \mathcal{N}(0,1)ϵ∼N(0,1)。因此，

    EN(z;μ,σ2)[f(z)]=EN(ϵ;0,1)[f(μ+σϵ)]≃1L∑l=1Lf(μ+σϵ(l))ϵ(l)∼N(0,1)\mathbb{E}_{\mathcal{N}(z;\mu,\sigma^2)}[f(z)] = \mathbb{E}_{\mathcal{N}(\epsilon;0,1)}[f(\mu+\sigma\epsilon)]\simeq \frac{1}{L}\sum_{l=1}^Lf(\mu + \sigma\epsilon^{(l)})\quad \epsilon^{(l)}\sim \mathcal{N}(0,1) E​N(z;μ,σ​2​​)​​[f(z)]=E​N(ϵ;0,1)​​[f(μ+σϵ)]≃​L​​1​​​l=1​∑​L​​f(μ+σϵ​(l)​​)ϵ​(l)​​∼N(0,1)

    那么我们应该如何选择可微分的变换gϕ(⋅)g_\phi(\cdot)g​ϕ​​(⋅)和辅助变量ϵ∼p(ϵ)\epsilon\sim p(\epsilon)ϵ∼p(ϵ)呢？三种基本方法为：

    1. 可追踪的逆CDF。在这种情况下，令ϵ∼U(0,I)\epsilon \sim \mathcal{U}(0,I)ϵ∼U(0,I)，令gϕ(ϵ,x)g_\phi(\epsilon, x)g​ϕ​​(ϵ,x)为qϕ(z∣x)q_\phi(z\mid x)q​ϕ​​(z∣x)的逆CDF。
    2. 与高斯分布的例子类似，对于任何"location-scale"族分布我们可以选择标准分布(location=0=0=0，scale=1=1=1)作为辅助分布ϵ\epsilonϵ，并且令$g(\cdot) = $location + scale ⋅ϵ\cdot \epsilon⋅ϵ。
    3. Composition：可以把随机变量表达成辅助变量的不同变换，利用对数正态。

    # Example: Variational Auto-Encoder

    我们将给出我们使用神经网络来作为概率编码器qϕ(z∣x)q_\phi(z\mid x)q​ϕ​​(z∣x)。我们假设隐变量的先验为中心各向同性的多元高斯分布pθ(z)=N(z;0,I)p_\theta(z) = \mathcal{N}(z;0,I)p​θ​​(z)=N(z;0,I)。注意在此情况下真正的后验分布pθ(z∣x)p_\theta(z\mid x)p​θ​​(z∣x)是不可追踪的。但是在qϕ(z∣x)q_\phi(z\mid x)q​ϕ​​(z∣x)的形式下我们有更多自由度，我们假设真正的后验分布取协方差矩阵近似对角矩阵的高斯分布。在这种情况下，我们可以让变分近似后验为协方差矩阵为对角矩阵的多元高斯分布：
