    # 马尔可夫决策过程

    • MDP是强化学习问题在数学上的理想化形式
    • MDP中的环境是完全可预测的
    • 几乎所有的强化学习问题都可以在数学上表示为马尔可夫决策过程

    # 马尔可夫过程

    马尔可夫性质：状态StS_tS​t​​具有马尔可夫性，当且仅当P[St+1∣St]=P[St+1∣S1,⋯,St]\mathbb{P}[S_{t+1}\mid S_t] = \mathbb{P}[S_{t+1}\mid S_1,\cdots,S_t]P[S​t+1​​∣S​t​​]=P[S​t+1​​∣S​1​​,⋯,S​t​​]。

    对于马尔可夫状态sss和其后继状态s′s^\primes​′​​，它们的状态转移概率定义为：

    Pss′=P[St+1=s′∣St=s]\mathcal{P}_{ss^\prime} = \mathbb{P}[S_{t+1} = s^\prime\mid S_t=s] P​ss​′​​​​=P[S​t+1​​=s​′​​∣S​t​​=s]

    马尔可夫过程可以分为三类：

    • 时间、状态都离散的马尔可夫过程：马尔可夫链
    • 时间连续、状态离散的马尔可夫过程：连续时间的马尔可夫链
    • 时间、状态都连续的马尔可夫过程

    马尔可夫过程由元组(S,P)(\mathcal{S,P})(S,P)构成。

    从初始状态S1=C1S_1=C_1S​1​​=C​1​​开始，我们可以从马尔可夫链中采样一些子序列，每个子序列又称为幕：S1,S2,⋯,STS_1,S_2,\cdots,S_TS​1​​,S​2​​,⋯,S​T​​。

    # 马尔可夫奖励过程

    马尔可夫奖励过程是具有价值的马尔可夫链。

    马尔可夫奖励过程，由元组(S,P,R,γ)(\mathcal{S,P,R,\gamma})(S,P,R,γ)

    • S\mathcal{S}S是有限状态的集合
    • P\mathcal{P}P是状态转移矩阵
    • Rs\mathcal{R}_sR​s​​是奖励函数，Rs=E[Rt+1∣St=s]\mathcal{R}_s = \mathbb{E}[\mathcal{R}_{t+1}\mid \mathcal{S}_t = s]R​s​​=E[R​t+1​​∣S​t​​=s]
    • γ∈[0,1]\gamma \in [0,1]γ∈[0,1]为折扣因子

    # 回报

    在一个马尔可夫奖励过程中，从ttt时刻的StS_tS​t​​开始，直到终止状态时，所有奖励的衰减之和GtG_tG​t​​称为回报。

    Gt=Rt+1+γRt+2+⋯=∑k=0∞γkRt+k+1G_t = R_{t+1} + \gamma R_{t+2} + \cdots = \sum_{k=0}^\infty \gamma^k R_{t+k+1} G​t​​=R​t+1​​+γR​t+2​​+⋯=​k=0​∑​∞​​γ​k​​R​t+k+1​​

    γ\gammaγ作用：

    • 避免有环的马尔可夫过程计算收益时出现无线循环
    • 从金融投资回报的角度讲，即时奖励比延迟奖励更有吸引力
    • 动物/人类行为中都表现出对即时奖励的偏好
    • 可以表达未来的不确定性
    • γ∈[0,1],γ=0\gamma \in [0,1],\gamma=0γ∈[0,1],γ=0只看眼前收益

    # 价值函数

    • 价值函数v(s)v(s)v(s)给出状态sss的长期价值
    • 价值函数输入为某个状态，输出为这个状态的价值。

    价值函数：

    在马尔可夫奖励过程中，一个状态的期望回报被称为这个状态的价值函数

    v(s)=E[Gt∣St=s]v(s) = \mathbb{E}[G_t\mid S_t=s] v(s)=E[G​t​​∣S​t​​=s]

    因为可能有不同的路径，存在不同的GtG_tG​t​​，另外奖励也可能存在随机性，需要消除随机性。

    # 贝尔曼方程

    贝尔曼方程，可以求解价值函数。

    贝尔曼方程

    v(s)=E[Rt+1+γv(St+1)∣St=s]v(s) = \mathbb{E}[R_{t+1}+\gamma v(S_{t+1})\mid S_t = s] v(s)=E[R​t+1​​+γv(S​t+1​​)∣S​t​​=s]

    价值函数可以分解为两部分

    • 即时奖励Rt+1R_{t+1}R​t+1​​
    • 后继状态的折扣值γv(St+1)\gamma v(S_{t+1})γv(S​t+1​​)

    推导过程：

    v(s)=E[Gt∣St=s]=E[Rt+1+γRt+2+γ2Rt+3+⋯∣St=s]=E[Rt+1+γ(Rt+2+γRt+3+⋯)∣St=s]=E[Rt+1+γGt+1∣St=s]=E[Rt+1+γv(St+1)∣St=s]\begin{aligned} v(s) &= \mathbb{E}[G_t\mid S_t = s]\\ &= \mathbb{E}[R_{t+1}+\gamma R_{t+2} + \gamma^2R_{t+3}+\cdots\mid S_t = s]\\ &= \mathbb{E}[R_{t+1}+\gamma(R_{t+2}+\gamma R_{t+3}+\cdots)\mid S_t =s]\\ &= \mathbb{E}[R_{t+1}+\gamma G_{t+1}\mid S_t=s]\\ &= \mathbb{E}[R_{t+1}+\gamma v(S_{t+1})\mid S_t=s] \end{aligned} ​v(s)​​​​​​​=E[G​t​​∣S​t​​=s]​=E[R​t+1​​+γR​t+2​​+γ​2​​R​t+3​​+⋯∣S​t​​=s]​=E[R​t+1​​+γ(R​t+2​​+γR​t+3​​+⋯)∣S​t​​=s]​=E[R​t+1​​+γG​t+1​​∣S​t​​=s]​=E[R​t+1​​+γv(S​t+1​​)∣S​t​​=s]​​

    贝尔曼方程还可以写作：

    v(s)=Rs+γ∑s′∈SPss′v(s′)v(s) = \mathcal{R}_s + \gamma \sum_{s^\prime \in \mathcal{S}}\mathcal{P}_{ss^\prime}v(s^\prime) v(s)=R​s​​+γ​s​′​​∈S​∑​​P​ss​′​​​​v(s​′​​)

    其中Rs=E[Rt+1∣St=s]\mathcal{R}_s = \mathbb{E}[R_{t+1}\mid S_t=s]R​s​​=E[R​t+1​​∣S​t​​=s]。

    只有当满足贝尔曼方程时我们的价值函数才叫价值函数。

    # 贝尔曼方程的矩阵形式
