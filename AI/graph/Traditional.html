    # 图机器学习的传统方法

    # 概述

    # 机器学习任务

    • 节点级别预测
    • 连接级别预测
    • 图级别预测

    # 传统机器学习的工作流

    我们假设我们的节点、连接和图都有特征与之对应。

    • 构建节点/连接/图的特征
    • 获取所有训练数据的特征

    一般的流程为：

    • 训练一个模型
      • 随机森林
      • SVM
      • 神经网络
    • 应用这个模型:
      • 给定一个新的节点/联系/图，得到它的特征并进行预测

    这节课我们将主要探究特征的设计。

    # 特征设计

    在图上使用高效的特征是达到好的表现的关键。传统的机器学习的途径使用手动产生的特征(hand-designed features)。 在本节课中，我们概述传统特征为：

    • Node-level prediction
    • Link-level prediction
    • Graph-level prediction 为了简便，我们只讨论无向图。

    # 图机器学习

    目标：对一系列对象(objects)进行预测。 设计的选择:

    • 特征: ddd-维向量
    • 对象: 节点, 边, 节点的集合, 整个图
    • 目标函数: 我们将要解决什么问题

    # 节点级别的任务

    看上面的这个例子，我们想要预测右边图中灰色的点的颜色是红色还是绿色。我们可以通过如下规则进行预测：绿色的点的度大于等于222,而红色点的度数为111。这样我们就选取了点的度数作为了特征，因此机器学习需要特征。

    机器学习需要特征。

    目标：描述网络中节点的结构和位置：

    • 节点的度
    • 节点的中心性(Node centrality)
    • 聚类系数(Clustering coefficient)
    • Graphlets

    # 连接级别的任务

    • 任务是在已经存在的连接的基础上预测新的连接
    • 在测试时，所有节点对(无存在的连接)被排序，并且最高的KKK个节点对被预测。
    • 关键是对一对节点设置特征。

    两种形式的连接预测任务：

    1. 连接随机消失(Links missing at random)：随机移除一系列连接然后预测它们
    2. Links over time：
      1. 给定直到t0′t_0^\primet​0​′​​时刻由边定义的图G[t0,t0′]G[t_0,t_0^\prime]G[t​0​​,t​0​′​​]，输出一个预测将要在G[t1,t1′]G[t_1,t_1^\prime]G[t​1​​,t​1​′​​]中出现的排序连接的列表LLL(不在G[t0,t0′]G[t_0,t_0^\prime]G[t​0​​,t​0​′​​]中)
      2. 计算
        1. n=∣Enew∣n=|E_{\text{new}}|n=∣E​new​​∣：为在测试期间[t1,t1′][t_1,t_1^\prime][t​1​​,t​1​′​​]出现的新边的个数
        2. 取LLL中高的nnn个元素并数正确的边。

    方法：

    • 对于每一个成对的节点(x,y)(x,y)(x,y)，计算分数c(x,y)c(x,y)c(x,y)
      • 例如：c(x,y)c(x,y)c(x,y)可以为节点x,yx,yx,y共同的相邻节点的数量
    • 通过c(x,y)c(x,y)c(x,y)递减的顺序对(x,y)(x,y)(x,y)进行排序
    • 预测最高的nnn对最为新的连接
    • 查看这些连接中哪些真正在G[t1,t1′]G[t_1,t_1^\prime]G[t​1​​,t​1​′​​]中出现
