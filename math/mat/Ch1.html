    # 第一章 线性空间与线性运算

    # 线性空间

    给定非空集合VVV和域F\mathbb{F}F，若存在映射σ\sigmaσ：

    V×V→V(V1,V2)↦σ(V1,V2)\begin{aligned} V\times V &\rightarrow V\\ (V_1,V_2)&\mapsto\sigma(V_1,V_2) \end{aligned} ​V×V​(V​1​​,V​2​​)​​​→V​↦σ(V​1​​,V​2​​)​​

    则称σ\sigmaσ为VVV上的加法。

    • 域：对+,−,×,÷+,-,\times,\div+,−,×,÷运算封闭的集合。
      • 有理数域：Q\mathbb{Q}Q
      • 实数域：R\mathbb{R}R
      • 复数域：C\mathbb{C}C
    • ×\times×集合的卡氏集：就是两个集合的笛卡尔积
    • →\rightarrow→表示映射，前后连接集合
    • ↦\mapsto↦也表示映射，但是前后连接元素

    V×F→V(v,x)↦σ(v,x)\begin{aligned} V\times \mathbb{F}&\rightarrow V\\ (v,x) &\mapsto \sigma(v,x) \end{aligned} ​V×F​(v,x)​​​→V​↦σ(v,x)​​

    称为VVV和F\mathbb{F}F之间的数乘法，且记为σ(v,x)=vx\sigma(v,x) = vxσ(v,x)=vx，且这两种运算满足通常的运算规则，则称集合VVV关于此加法和数乘法是域F\mathbb{F}F的线性空间。有时也会把线性空间称为向量空间。

    • 通常的运算规则：
      • 加法
        • 加法交换律：V1+V2=V2+V1V_1+V_2 = V_2+V_1V​1​​+V​2​​=V​2​​+V​1​​
        • 加法的结合律：(V1+V2)+V3=V1+(V2+V3)(V_1+V_2)+V_3 = V_1+(V_2+V_3)(V​1​​+V​2​​)+V​3​​=V​1​​+(V​2​​+V​3​​)
        • 有零元，存在e∈V,e+v=ve\in V,e+v = ve∈V,e+v=v
        • 有负元：对任意v∈Vv\in Vv∈V，存在a∈Va\in Va∈V，使a+v=ea + v = ea+v=e，记a=−va = -va=−v
      • 数乘法
        • 分配律：(v1+v2)⋅k=v1⋅k+v2⋅k(v_1+v_2)\cdot k = v_1\cdot k+v_2\cdot k(v​1​​+v​2​​)⋅k=v​1​​⋅k+v​2​​⋅k
        • v⋅(k1+k2)=v⋅k1+v⋅k2v\cdot(k_1+k_2) = v\cdot k_1 + v\cdot k_2v⋅(k​1​​+k​2​​)=v⋅k​1​​+v⋅k​2​​：在这个运算中两个加法含义不同：
          • 等号左边的加法是数域F\mathbb{F}F中的加法
          • 等号右边的加法是我们新定义的VVV上的加法
        • 与F\mathbb{F}F中元素的关系
          • v⋅(kl)=(v⋅k)lv\cdot(kl) = (v\cdot k)lv⋅(kl)=(v⋅k)l
            • 等号左边先是数域F\mathbb{F}F中普通的乘法，后是VVV与F\mathbb{F}F的数乘法。
            • 等号左边相当于两次数乘法
          • 与F\mathbb{F}F中111的关系：V⋅1=VV\cdot1 = VV⋅1=V

    数乘法的数字在左边或右边

    因为将数字写在右边就可以看作矩阵的乘法，列向量vvv为n×1n\times1n×1，数字kkk看成1×11\times11×1，符合向量乘法。

    # 向量组的极大线性无关子组

    子序列：从母序列中挑出一个子序列构成向量组，这个子序列构成的向量组是原来母序列的子组。

    β1,β2,⋯,βs\beta_1,\beta_2,\cdots,\beta_sβ​1​​,β​2​​,⋯,β​s​​为向量组α1,α2,⋯,αp\alpha_1,\alpha_2,\cdots,\alpha_pα​1​​,α​2​​,⋯,α​p​​的子组，其为最大线性无关子组若：

    • {βj}\{\beta_j\}{β​j​​}线性无关
    • 若存在另一个子组{γj}t\{\gamma_j\}_t{γ​j​​}​t​​，若t>st>st>s(即其数目大)，则{γj}t\{\gamma_j\}_t{γ​j​​}​t​​线性相关。

    定理：向量组的极大线性无关子组所含向量的的个数是惟一的。

    证明：假设向量组[α1,⋯,αp]=A[\alpha_1,\cdots,\alpha_p]=A[α​1​​,⋯,α​p​​]=A有两个极大线性无关组，[β1,⋯,βs]=B[\beta_1,\cdots,\beta_s]=B[β​1​​,⋯,β​s​​]=B和[γ1,⋯,γt]=C[\gamma_1,\cdots,\gamma_t]=C[γ​1​​,⋯,γ​t​​]=C。假设s>ts>ts>t。易知BBB可由AAA线性表示，AAA能由CCC线性表示，即AX=B,CY=AAX = B, CY=AAX=B,CY=A，所以有B=CYXB=CYXB=CYX。而Z=YXZ=YXZ=YX为t×st\times st×s的矩阵，又因为s>ts>ts>t，所以Zw=0Zw=0Zw=0肯定有非零解，所以Bw=CZw=0Bw = CZw = 0Bw=CZw=0，即存在不全为零的www使得[β1,⋯,βs]=B[\beta_1,\cdots,\beta_s]=B[β​1​​,⋯,β​s​​]=B的线性组合为零，所以BBB线性相关，矛盾。
