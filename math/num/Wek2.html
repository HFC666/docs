    # Sensitivity of a linear system

    给定

    Ax=b\mathrm{Ax=b} Ax=b

    解为：

    x^=A−1b\hat{\mathrm{x}} = \mathrm{A^{-1}b} ​x​^​​=A​−1​​b

    • 只是近似地知道A\mathrm{A}A
    • 近似地知道b\mathrm{b}b

    我们只能得到一个近似解x⋆\mathrm{x}^\starx​⋆​​

    令x⋆\mathrm{x}^\starx​⋆​​为Ax=b\mathrm{Ax=b}Ax=b的近似解，我们想要去衡量x⋆\mathrm{x}^\starx​⋆​​和x^\mathrm{\hat{x}}​x​^​​的差异：

    • 偏差e=x^−x⋆\mathrm{e = \hat{x}-\mathrm{x}^\star}e=​x​^​​−x​⋆​​
    • 残差r=b−Ax⋆=Ae\mathrm{r=b-Ax^\star=Ae}r=b−Ax​⋆​​=Ae

    所以我们需要衡量向量之间的距离。

    def householder(vec):
    """Construct a Householder reflection to zero out 2nd and further components of a vector.

    Parameters
    ----------
    vec : array-like of floats, shape (n,)
        Input vector
    
    Returns
    -------
    outvec : array of floats, shape (n,)
        Transformed vector, with ``outvec[1:]==0`` and ``|outvec| == |vec|``
    H : array of floats, shape (n, n)
        Orthogonal matrix of the Householder reflection
    """
    vec = np.asarray(vec, dtype=float)
    if vec.ndim != 1:
        raise ValueError("vec.ndim = %s, expected 1" % vec.ndim)
    # ... ENTER YOUR CODE HERE ...
    outvec = np.zeros(len(vec))
    outvec[0] = np.linalg.norm(vec)
    u = np.zeros(len(vec))
    u[0] = -np.sum(vec[1:]**2) / (vec[0] + outvec[0])
    u[1:] = vec[1:]
    u = u / np.linalg.norm(u)
    H = np.eye(len(u)) - 2 * np.outer(u, u)
    return np.dot(H,vec),H
