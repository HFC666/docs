    # 第一章 集合及其运算

    # 集合的概念

    一些可以区分的，互不相同的东西构成的整体称为集合。关于定义中的东西不太严谨，我们采用元素。 这样我们就有集合和元素了，集合和元素的关系可以分为属于(∈\in∈)和不属于(∉\notin∉)。 集合的表示方法：枚举法、概述集合中元素的性质(描述法)。 集合的分类：

    • 无穷集合
    • 有穷集合
    • 空集合

    # 子集、集合的相等

    设A,BA,BA,B为集合，如果AAA中的每个元素都属于BBB，则称AAA为BBB的子集，记作A⊆BA\subseteq BA⊆B。

    A⊆B⇔∀x∈A→x∈BA\subseteq B \Leftrightarrow \forall x\in A\rightarrow x\in B A⊆B⇔∀x∈A→x∈B

    也

    x∉B→x∉Ax\notin B\rightarrow x\notin A x∉B→x∉A

    那么对于A⊈B⇔∃x∈A,x∉BA\nsubseteq B\Leftrightarrow \exists x\in A,x\notin BA⊈B⇔∃x∈A,x∉B。

    真子集：A⊂B⇔B,∃x∈BA\subset B\Leftrightarrow B, \exists x\in BA⊂B⇔B,∃x∈B使x∉Ax\notin Ax∉A。

    显然会有下面的结论成立：

    1. A⊆AA\subseteq AA⊆A
    2. A⊆B,B⊆CA\subseteq B,B\subseteq CA⊆B,B⊆C，则A⊆CA\subseteq CA⊆C
    3. ∅⊆A\varnothing \subseteq A∅⊆A

    下面我们证明一下第三个： 我们可以其逆否命题成立：不属于AAA的元素一定不属于∅\varnothing∅；也可以直接进行证明：如果x∈∅x\in \varnothingx∈∅则x∈Ax\in Ax∈A。这个命题由于前提是错误的，所以此命题为真命题。

    设A,BA,BA,B为集合，如果A⊆BA\subseteq BA⊆B且B⊆AB\subseteq AB⊆A，则AAA与BBB相等，即为A=BA=BA=B。

    显然我们可以得到以下性质：

    • A=AA=AA=A
    • 如果A=BA=BA=B，则B=AB=AB=A
    • 如果A=B,B=CA=B,B=CA=B,B=C，则A=CA=CA=C

    集族：集合的集合 幂集：设AAA为集合，P(A)={B∣B⊆A}\mathcal{P}(A)=\{B\mid B\subseteq A\}P(A)={B∣B⊆A}，我们也可以将其记为2A2^A2​A​​，这是因为∣2A∣=2∣A∣|2^A|=2^{|A|}∣2​A​​∣=2​∣A∣​​。空集的幂集为{∅}\{\varnothing\}{∅}，并且∅∈2∅,∅⊆2∅\varnothing \in 2^{\varnothing},\varnothing \subseteq 2^{\varnothing}∅∈2​∅​​,∅⊆2​∅​​。 下面我们讲一下康拓悖论：假设MMM为所有集合的集合，那么2M⊆M2^M\subseteq M2​M​​⊆M，即∣2M∣≤∣M∣|2^M|\le |M|∣2​M​​∣≤∣M∣，但是根据康托证明的定理我们有∣M∣<∣2M∣|M|<|2^M|∣M∣<∣2​M​​∣。相矛盾。

    # 集合的运算

    # 并运算

    设A,BA,BA,B为集合，由属于AAA或属于BBB的一切元素组成的集合称为AAA与BBB的并，记作A∪BA\cup BA∪B。

    A\cup B=\{x\mid x\in A\text{或}x\in B\}

    所以我们有

    \begin{aligned} x\in A\cup B&\Leftrightarrow x\in A\text{或}x\in B\\ x\notin A\cup B&\Leftrightarrow x\notin A\text{且}x\notin B \end{aligned}

    定理1：假设A,B,CA,B,CA,B,C为集合，则

    1. A∪A=AA\cup A=AA∪A=A
    2. A∪B=B∪AA\cup B=B\cup AA∪B=B∪A
    3. (A∪B)∪C=A∪(B∪C)(A\cup B)\cup C = A\cup (B\cup C)(A∪B)∪C=A∪(B∪C)
    4. ∅∪A=A\varnothing \cup A=A∅∪A=A
    5. A∪B=B⇔A⊆BA\cup B=B\Leftrightarrow A\subseteq BA∪B=B⇔A⊆B

    设A1,A2,⋯,AnA_1,A_2,\cdots,A_nA​1​​,A​2​​,⋯,A​n​​为集合，它们的并记为

    ⋃i=1nAi=A1∪A2∪⋯∪An\bigcup_{i=1}^nA_i=A_1\cup A_2\cup \cdots \cup A_n ​i=1​⋃​n​​A​i​​=A​1​​∪A​2​​∪⋯∪A​n​​

    所以我们有：

    x∈⋃i=1nAi⇔∃i,1≤i≤n,x∈Aix\in \bigcup_{i=1}^nA_i \Leftrightarrow \exists i,1\le i\le n,x\in A_i x∈​i=1​⋃​n​​A​i​​⇔∃i,1≤i≤n,x∈A​i​​

    # 交运算

    定义：假设A,BA,BA,B是集合，由属于AAA并且属于BBB的元素构成的集合称为AAA与BBB的并，记为A∩BA\cap BA∩B。

    A\cap B = \{x\mid x\in A\text{且}x\in B\}

    所以我们有

    \begin{aligned} x\in A\cap B&\Leftrightarrow x\in A\text{且}x\in B\\ x\notin A\cap B&\Leftrightarrow x\notin A\text{或}x\notin B \end{aligned}

    定理2：设A,B,CA,B,CA,B,C为集合：

    1. A∩A=AA\cap A=AA∩A=A
    2. A∩B=B∩AA\cap B=B\cap AA∩B=B∩A
    3. ∅∩A=∅\varnothing\cap A=\varnothing∅∩A=∅
    4. A∩(B∩C)=(A∩B)∩CA\cap(B\cap C)=(A\cap B)\cap CA∩(B∩C)=(A∩B)∩C
    5. A∩B=A⇔A⊆BA\cap B=A\Leftrightarrow A\subseteq BA∩B=A⇔A⊆B

    设A1,A2,⋯,AnA_1,A_2,\cdots,A_nA​1​​,A​2​​,⋯,A​n​​为集合，

    ⋂i=1nAi=A1∩A2∩⋯∩An\bigcap_{i=1}^nA_i = A_1\cap A_2\cap \cdots\cap A_n ​i=1​⋂​n​​A​i​​=A​1​​∩A​2​​∩⋯∩A​n​​

    所以我们有

    x∈⋂i=1nAi⇔∀i,x∈Aix\in \bigcap_{i=1}^nA_i\Leftrightarrow \forall i, x\in A_i x∈​i=1​⋂​n​​A​i​​⇔∀i,x∈A​i​​

    如果Ai∩Aj=∅A_i\cap A_j=\varnothingA​i​​∩A​j​​=∅，则称AiA_iA​i​​与AjA_jA​j​​不相交。如果∀i,j∈{1,⋯,N},i≠j,Ai∩Aj=∅\forall i,j\in \{1,\cdots,N\},i\neq j,A_i\cap A_j=\varnothing∀i,j∈{1,⋯,N},i≠j,A​i​​∩A​j​​=∅，则称AiA_iA​i​​为两两不相交的集序列。

    现在我们有两个运算了，现在讨论一下分配律： 定理3：

    A∩(⋃ξ∈IAξ)=⋃ξ∈I(A∩Aξ)A∪(⋂ξ∈IAξ)=⋂ξ∈I(A∪Aξ)\begin{aligned} A\cap\left(\bigcup_{\xi\in I}A_{\xi}\right) = \bigcup_{\xi\in I}(A\cap A_{\xi})\\ A\cup\left(\bigcap_{\xi\in I}A_{\xi}\right)=\bigcap_{\xi\in I}(A\cup A_{\xi}) \end{aligned} ​A∩​⎝​⎛​​​ξ∈I​⋃​​A​ξ​​​⎠​⎞​​=​ξ∈I​⋃​​(A∩A​ξ​​)​A∪​⎝​⎛​​​ξ∈I​⋂​​A​ξ​​​⎠​⎞​​=​ξ∈I​⋂​​(A∪A​ξ​​)​​

    我们可以通过集合相等的定义来证明。 根据定理3，我们可以得到定理4：

    • A∩(B∪C)=(A∩B)∪(A∩C)A\cap (B\cup C)=(A\cap B)\cup(A\cap C)A∩(B∪C)=(A∩B)∪(A∩C)
    • A∪(B∩C)=(A∪B)∩(A∪C)A\cup (B\cap C)=(A\cup B)\cap (A\cup C)A∪(B∩C)=(A∪B)∩(A∪C)
    • A∩(A∪B)=AA\cap (A\cup B)=AA∩(A∪B)=A
    • A∪(A∩B)=AA\cup (A\cap B)=AA∪(A∩B)=A

    例子：跳舞问题，假设在毕业舞会上，有男有女，每个男生都与至少一个女生跳过舞，但是并没有跟所有女生跳过舞，每个女生至少跟一个男生跳过舞，但是没有跟所有男生跳过舞。证明存在两男两女，这两个男生中的每一个只与这两个女生中的一个跳过舞，这两个女生中每一个也只与这两个男生中的一个跳过舞。

    # 差运算

    设A,BA,BA,B是两个集合，有属于AAA中但是不属于BBB的元素的集合称为AAA与BBB的差。

    A\B={x∈A,x∉B}A \backslash B = \{x\in A, x\notin B\} A\B={x∈A,x∉B}

    存在性质：

    A∩(B\C)=(A∩B)\(A∩C)A\cap (B \backslash C) = (A\cap B) \backslash (A\cap C) A∩(B\C)=(A∩B)\(A∩C)

    # 对称差

    设A,BA,BA,B是两个集合，则(A\B)∪(B\A)(A\backslash B) \cup (B\backslash A)(A\B)∪(B\A)为A,BA,BA,B的对称差，记作AΔBA\Delta BAΔB。

    性质：

    • AΔB=BΔAA\Delta B = B\Delta AAΔB=BΔA
    • AΔ(BΔC)=(AΔB)ΔCA\Delta (B\Delta C) = (A\Delta B)\Delta CAΔ(BΔC)=(AΔB)ΔC
    • AΔA=∅A\Delta A = \varnothingAΔA=∅
    • AΔ∅=AA\Delta \varnothing =AAΔ∅=A
    • A∩(BΔC)=(A∩B)Δ(A∩C)A\cap (B\Delta C) = (A\cap B)\Delta (A\cap C)A∩(BΔC)=(A∩B)Δ(A∩C)
